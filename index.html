Abstract

Research Overview

This research investigates a privacy-preserving framework for the classification of bacterial, fungal, viral, and parasitic skin diseases using federated learning and transfer learning. Each participating client institution trains a MobileNetV2-based model locally on dermatological images, then shares model updates rather than raw patient data.

The aggregated global model improves diagnostic performance while reducing central data exposure, making the system suitable for distributed healthcare settings that require data confidentiality, scalable deployment, and clinically useful prediction speeds.
